# String Permutation

You are given two string arrays. Write a program to check if they are permutations of each other.

---

__Examples__

```java
String[] array1 = {"Hello", "hey", "hi", "hola", "bye"};

String[] array2 = {"hi", "hey", "Hello", "bye", "hola"};

permutation(array1, array2) // true
```
